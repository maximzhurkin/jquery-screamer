Screamer

Very simple jQuery plugin for custom alerts

Download plugin
  • Default
  • Click overlay for close
  • Custom button text
  • With title

1. Include styles

<link rel="stylesheet" href="screamer.jquery.min.css">
<link rel="stylesheet" href="screamer.theme.min.css">

2. Include jQuery and Screamer

<script src="jquery.min.js"></script>
<script src="screamer.jquery.min.js"></script>

3. Call Screamer

$('#alert').click(function(){ $().screamer({ message: 'Its default alert' }); });
